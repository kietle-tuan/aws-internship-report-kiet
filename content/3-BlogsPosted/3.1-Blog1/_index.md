# INTRODUCTION TO AWS CLOUD COMPUTING

## 1. Overview

This blog summarizes what I learned about basic cloud computing and Amazon Web Services during the internship.

AWS Cloud Computing was a new topic for me at the beginning, so I mainly focused on understanding basic concepts through tutorials, AWS Study Group materials, and YouTube videos.

The purpose of this blog is to record my basic understanding of cloud computing and how AWS services can support web applications.

---

## 2. What Is Cloud Computing?

Cloud computing allows users to use computing resources through the internet instead of managing physical servers directly.

These resources may include:

- Servers
- Storage
- Databases
- Networking
- Security services
- Monitoring tools
- Application deployment services

With cloud computing, developers can build and run applications more flexibly without needing to own and maintain all physical infrastructure.

---

## 3. What Is AWS?

**Amazon Web Services (AWS)** is a cloud computing platform that provides many services for building, deploying, and managing applications.

AWS services can support different parts of a system, such as computing, storage, networking, database management, security, and monitoring.

During the internship, I learned AWS at a basic level and focused on understanding the purpose of common services.

---

## 4. Basic AWS Services I Learned

| AWS Service | Basic Purpose |
|---|---|
| IAM | Manage users, roles, and permissions |
| EC2 | Provide virtual servers for running applications |
| S3 | Store files and objects |
| VPC | Create and manage a cloud network environment |
| Security Groups | Control inbound and outbound traffic |
| CloudWatch | Monitor logs, metrics, and system behavior |

These services helped me understand how a cloud-based application can be organized.

---

## 5. Why AWS Is Useful

AWS is useful because it provides cloud resources that can support different types of applications.

Some benefits include:

- Users can create cloud resources when needed.
- Applications can be deployed without managing physical servers directly.
- Storage and computing resources can be adjusted more flexibly.
- Monitoring tools help observe application behavior.
- Security services help control access to cloud resources.

At my current level, I understood these benefits in a basic way through tutorials and learning materials.
